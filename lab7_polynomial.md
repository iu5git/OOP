# 7 - Работа с полиномами (классы Term и Polynomial)

## Цель работы

- Разработать классы для представления полинома.
- Освоить перегрузку операторов.
- Закрепить навыки работы с динамической памятью.
- Научиться реализовывать арифметические операции над объектами.

## Задание

Необходимо реализовать два класса:

- `Term` — одночлен;
- `Polynomial` — полином.

---

## Класс `Term`

### Общая характеристика

Класс представляет одночлен вида:

ax^n

### Поля класса

- `coef` — коэффициент;
- `power` — степень.

### Методы

- Конструкторы;
- Операторы сравнения;
- Оператор вывода.

---

## Класс `Polynomial`

### Общая характеристика

Полином — это сумма одночленов.

### Требования

- Хранение в виде динамического массива или списка;
- Автоматическое упрощение (объединение одинаковых степеней);
- Сортировка по убыванию степеней.

### Методы

- Добавление одночлена;
- Удаление одночлена;
- Приведение подобных членов;
- Вычисление значения при x.

### Операторы

- `+` — сложение;
- `-` — вычитание;
- `*` — умножение;
- `==` — сравнение;
- `<<` — вывод.

---

## Пример

```cpp
Polynomial p1, p2;

p1.add(Term(2, 2)); // 2x^2
p1.add(Term(3, 1)); // 3x

p2.add(Term(1, 2)); // x^2

Polynomial res = p1 + p2;

std::cout << res << std::endl;
```
